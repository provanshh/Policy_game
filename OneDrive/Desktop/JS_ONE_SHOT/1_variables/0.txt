- variables are used to store data (form of container)

- var(oldest) , let(latest) , const 

- variables can store any type of data (string, list, arrays, function)

- variables name cannot begin with number

- JS is a case sensitive language

- let keyword : used to declare a blocked scope variable
                variable is only visible within the block in which its declared

- const keyword : used to declare a constant variable
                  variable cannot be reassigned to a new value
